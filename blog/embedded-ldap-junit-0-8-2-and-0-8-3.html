The embedded-ldap-junit project provides convenient access to an embedded LDAP server (the in-memory directory server provided by the UnboundID LDAP SDK for Java) for use in JUnit unit tests. They have just released versions 0.8.2 and 0.8.3 of the library, which appear to make it easier to use LDAPS (version 0.8.2) and to set a server-side search size limit (version 0.8.3).

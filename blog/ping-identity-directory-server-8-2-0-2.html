Ping Identity Directory Server version 8.2.0.2 has just been released. As noted in the release notes, the changes over the 8.2.0.1 release include:

  • Expanded single sign-on support for the Admin Console. It was previously only supported in conjunction with the PingOne service, but it can now be used with any OpenID Connect provider. It also offers additional options for determining whether to trust the TLS certificate chain presented by the OpenID Connect provider.
  • Improved performance when requesting a conflict prevention details entry through the uniqueness request control.

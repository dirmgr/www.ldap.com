Although the website hasn’t yet been updated to reflect it, the 389 Directory Server project has announced the release of the 1.4.1.2 and 1.4.0.22 versions. The 1.4.0.22 version looks like it’s primarily a bugfix release, addressing problems in areas like memory management, caching, replication, LDIF export, administrative interfaces, and other areas. The 1.4.1.2 release looks like it includes all of the fixes from 1.4.0.22, plus additional fixes, additional tests, build process updates, improved internal operation logging, and tooling updates.

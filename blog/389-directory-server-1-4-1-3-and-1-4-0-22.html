The 389 Directory Server project has announced new 1.4.1.3 and 1.4.0.23 versions. The 1.4.0.23 version appears to be a bugfix release (including a couple of potential security-related issues) and updates to administrative interfaces. The 1.4.1.3 release includes additional fixes (including potential crashes), improves logging for internal operations, and adds support for compare operations in the configuration.

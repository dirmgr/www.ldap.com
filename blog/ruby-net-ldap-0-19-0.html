The ruby-net-ldap project provides LDAP support for the Ruby programming language. They have just released version 0.19.0 of the library. Some of the changes in this release appear to include:

  • Fixed an OpenSSL-related error when using multiple hosts
  • Added the ability to use controls when performing searches
  • Fixed an issue in which DNs could inappropriately drop trailing spaces in RDN values
  • Added missing documentation for a connect_timeout property

Ping Identity Directory Server version 8.2.0.1 has just been released. As noted in the release notes, the changes over the 8.2.0.0 release include:

  • Updated the purge expired data plugin to support multiple concurrent threads
  • Fixed an issue that affected compatibility with the Delegated Admin app
  • Fixed an issue that could prevent some Server SDK plugins targeting add operations from seeing some other types of changes made to that add operation internally within the server

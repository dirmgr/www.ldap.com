GLAuth is a simple LDAP server that positions itself as a lightweight alternative to OpenLDAP or Active Directory. The project has just released version 2.2.0. Some of the changes in this release appear to include:

  • Don't attempt to log to syslog on Windows
  • When constructing DNs for the uniqueMember attribute, it will automatically place them under ou=Users
  • Use NameFormat/GroupFormat when constructing RDN attributes
  • Support navigating up and down the tree with LDAP browsers
  • Fixed an issue in which a search to identify POSIX accounts could ignore the search base DN
  • Updated the configuration sample to allow anonymous access to the root DSE

Symas has released version 2.4.57.1 of their OpenLDAP Gold product, which is an enhanced version of OpenLDAP with additional features and commercial support. According to the release announcement and the release notes, it looks like the changes in this release include:

  • Fixed various memory management issues
  • Fixed an issue in which a cancel operation may not correctly terminate
  • Fixed synchronization processing to ignore duplicate sessionlog entries
  • Fixed issues around certificate validity checks
  • Fixed issues around sorting values in an invalid RDN
  • Updated the ldapextop tool to use the correct return code
  • Updated the client library to fix issues with malformed or maliciously crafted packets
  • Updated the client library to fix an issue with binds when re-establishing a closed connection
  • Updated the OpenSSL library to version 1.1.1i

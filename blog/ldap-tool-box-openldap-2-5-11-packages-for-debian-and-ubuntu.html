The LDAP Tool Box project offers a number of tools, scripts, and other niceties for working with LDAP. They have announced the release of OpenLDAP 2.5.11 packages for Debian and Ubuntu. OpenLDAP version 2.5.11 is essentially the same as OpenLDAP version 2.5.10 but with a build problem fixed, so you can see the OpenLDAP 2.6.1 and 2.5.10 release announcement to see what has changed.

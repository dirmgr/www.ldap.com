The 389 Directory Server project has announced new releases of versions 1.4.4.0, 1.4.3.6, 1.4.2.12, and 1.4.1.18. From the release announcements, it looks like some of the changes in these versions are:

  • Fixed a potential buffer overflow (versions 1.4.2.12, 1.4.3.6, and 1.4.4.0)
  • Fixed memory leaks in changelog code (versions 1.4.3.6 and 1.4.4.0)
  • Fixed an issue in which there may be intermittent hangs in TLS communication (all four versions)
  • Fixed an issue in which setting nsslapd-allowed-sasl-mechanisms could cause the value to be truncated (all four versions)

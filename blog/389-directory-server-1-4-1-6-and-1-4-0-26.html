The 389 Directory Server project has announced new 1.4.1.6 and 1.4.0.26 versions. The 1.4.0.26 version fixes an issue with setting the minimum and maximum SSL protocol versions and adds a security tab to the UI. The 1.4.1.6 version includes those changes, and also adds support for upgrading a password hash when performing a bind, updates a memory allocation library, among other things.

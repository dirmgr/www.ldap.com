The 389 Directory Server project has released version 1.4.3.13. Some of the changes in this release include:

  • Fixed an issue that could cause ds-replcheck to crash in offline mode.
  • Fixed an issue in which the SyncRepl plugin could provide an incorrect cookie value.
  • Fixed an issue that could prevent the processing start time from being set fro extended operations.
  • Fixed an issue that could affect searchrate performance

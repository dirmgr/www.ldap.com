The 389 Directory Server project has announced the release of version 1.4.3.21. From the release announcement, it looks like some of the changes in these versions are:

  • Fixed a crash that could occur when trying to replicate a modify DN operation
  • Fixed a changelog caching issue
  • Fixed a UI issue that could prevent it from showing configuration updates
  • Updated the server to log a message if a persistent search causes a connection thread limit to be exceeded
